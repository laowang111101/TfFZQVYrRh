## 项目概述 java444

本项目是一款基于Vue3的简单商城系统后台管理平台。它提供了用户登录、购物车以及商品管理等功能，支持下单流程。此项目作为纯前端无后台的Vue3项目作业，完全采用Vue3的setup语法糖进行编写。

## 技术栈

- **项目框架：** `Vue3` + `Vite`
- **路由管理：** `Vue Router 4+`
- **数据请求：** `axios`
- **UI实现：** `Element Plus`
- **状态管理：** `Pinia`

## 功能模块

- **用户模块：**
  - 登录注册
- **商品管理：**
  - 商品信息展示
- **购物车管理：**
  - 添加/删除商品
- **订单管理：**
  - 下单流程

## 系统角色

- **普通用户：** 可以浏览商品，添加商品到购物车，并进行下单操作。

## 运行环境

- Node.js环境
- npm包管理工具

## 部署步骤

1. 克隆项目代码到本地
2. 执行 `npm i` 安装依赖
3. 执行 `npm run dev` 启动项目

## 核心亮点

- **现代化前端技术栈：** 项目基于Vue3，采用了最新的前端工具和库，确保了项目的现代性和可维护性。
